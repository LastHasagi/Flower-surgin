# Flower-surgin

Projeto em HTML, CSS e JavaScript: mini-game platformer retrô para a Agatha, com música 8-bit (Web Audio), coleta de 5 flores com mensagens e, no final, a animação de flores em CSS com o pedido de namoro.

## Como abrir

Abra `index.html` no navegador (duplo clique ou sirva com um servidor local). Em mobile, use os botões na parte inferior; no PC, setas/WASD e Espaço/W para pular.

## Arquivos

- `index.html` — estrutura do jogo, overlays e flores
- `styles.css` — flores animadas + UI do jogo
- `game.js` — engine do platformer, áudio e estados
- `script.js` — transição para as flores, pedido e confete
